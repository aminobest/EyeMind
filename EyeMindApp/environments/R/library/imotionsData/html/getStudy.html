getStudy {imotionsData}R Documentation

Get study by id

Description

Retrieves detailed information about a study including stimuli, respondents, segments, and more.

Usage

getStudy(connection, studyId)

Arguments

connection

As returned from connect function.

studyId

The study

Details

Available studies can be found with imotionsData::listStudies

Value

A data frame containing information about the Study

Examples

studies <- imotionsData::listStudies(connection)
studyId <- studies$id[1]
study <- imotionsData::getStudy(connection, studyId)
