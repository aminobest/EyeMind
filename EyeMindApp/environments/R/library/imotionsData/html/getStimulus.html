getStimulus {imotionsData}R Documentation

Get stimulus information

Description

Retrieves detailed information about a stimulus

Usage

getStimulus(connection, studyId, stimulusId)

Arguments

connection

As returned from connect function.

studyId

The study in which to search for stimulus

stimulusId

The stimulus to get information for

Details

Available studies can be found with imotionsData::listStudies.

Available stimuli can be found with imotionsData::listStimuli.

Value

A data frame containing information about the stimulus

Examples

studies <- imotionsData::listStudies(connection)
studyId <- studies$id[1]
stimuli <- imotionsData::listStimuli(connection, studyId)
stimulus <- imotionsData::getStimulus(connection, studyId, stimuli$id[1])
