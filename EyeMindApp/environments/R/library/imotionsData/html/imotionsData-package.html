imotionsData-package {imotionsData}R Documentation

iMotions Data Library

Description

A client for accessing data from the iMotions Biometrics Research Platform.

Details

Use tokens (found on a study's R Analysis page) to load information from the iMotions API

See Also

Useful links:

  • https://imotions.com

  • https://my.imotions.com/#studies

Examples

myToken <- "xxxxx-xxxx-xxxxx-xxxxx"
connection <- imotionsData::connectRemote(myToken)
studies <- imotionsData::listStudies(connection)
