privateDownloadData {imotionsApi}R Documentation

Return metrics / signals (with modified Timestamp if needed) for the sensor of interest

Description

Return metrics / signals (with modified Timestamp if needed) for the sensor of interest

Usage

privateDownloadData(study, sensor, signalsName = NULL)

Arguments

study

An imStudy object as returned from imStudy.

sensor

An imSensor object as returned from getRespondentSensors.

signalsName

Optional: A vector of specific signals name you would like to return.

Value

a data.frame with all signals (or specified signals) from the sensor of interest
