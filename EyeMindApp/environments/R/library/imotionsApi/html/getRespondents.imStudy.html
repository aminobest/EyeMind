getRespondents.imStudy {imotionsApi}R Documentation

Get all respondents from a study.

Description

S3 method default method to get all respondents for a study.

Usage

## S3 method for class 'imStudy'
getRespondents(study, ...)

Arguments

study

An imStudy object as returned from imStudy.

...

Further arguments passed from the getRespondents method.

Value

An imRespondentList object (data.table) with all respondents for the study.
