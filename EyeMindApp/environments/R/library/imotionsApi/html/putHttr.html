putHttr {imotionsApi}R Documentation

Perform a PUT HTTP request with authentication.

Description

The request will be retried up to 3 times if an error is encountered.

Usage

putHttr(connection, url, reqBody)

Arguments

connection

An imConnection object as returned from imConnection.

url

The url/path where the JSON file is located.

reqBody

The body of the request.

Value

The last HTTP response.
