getStimuli {imotionsApi}R Documentation

Get all stimuli from a study.

Description

Retrieves detailed information about stimuli in the study.

Usage

getStimuli(study, relevant = TRUE)

Arguments

study

An imStudy object as returned from imStudy.

relevant

A boolean indicating whether only relevant stimuli should be kept, by default non-relevant stimuli are discarded.

Value

An imStimulusList object (data.table) containing all stimuli from the study.

Examples

## Not run: 
connection <- imotionsApi::imConnection("xxxxxxxx")
studies <- imotionsApi::listStudies(connection)
study <- imotionsApi::imStudy(connection, studies$id[1])
stimuli <- imotionsApi::getStimuli(study)

## End(Not run)
